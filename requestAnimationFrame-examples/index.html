requestAnimationFrame 示例集

requestAnimationFrame 是浏览器提供的一个用于优化动画效果的 API，它会告诉浏览器你希望执行一个动画，并请求浏览器在下一次重绘之前调用指定的回调函数来更新动画。

以下示例展示了 requestAnimationFrame 在不同场景下的应用。点击链接可以查看具体示例和详细说明。

你还可以查看 面试题集锦 来了解更多相关知识点。

  • 1. 基础动画

    展示了 requestAnimationFrame 的基本用法，实现平滑的元素移动动画。与 setTimeout/setInterval 不同，requestAnimationFrame 会自动与浏览器的重绘周期同步，提供更平滑的动画体验。

  • 2. Canvas 动画

    展示了 requestAnimationFrame 在 Canvas 绘图中的应用。Canvas 是 requestAnimationFrame 的理想搭配，因为每一帧都需要清除画布并重新绘制，与浏览器的重绘周期同步可以获得最佳性能。

  • 3. 平滑滚动

    展示了如何使用 requestAnimationFrame 实现平滑的页面滚动效果。相比 CSS 的 scroll-behavior: smooth，这种方法提供了更多的控制权和自定义的动画曲线。

  • 4. 事件节流

    展示了如何使用 requestAnimationFrame 优化滚动、鼠标移动等高频率事件的处理。相比传统的 setTimeout/setInterval 节流方法，这种方式更加高效。
